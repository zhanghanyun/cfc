# Cloudflare ip 优选
### 使用 httping 测试 cloudflare ips，选取延迟最低的 ip，自动修改系统 hosts 文件，无需任何手动操作
## Usage
在 [release](https://github.com/zhanghanyun/cfc/releases) 下载相应文件解压运行

需要管理员权限运行（修改系统 Hosts 文件）
